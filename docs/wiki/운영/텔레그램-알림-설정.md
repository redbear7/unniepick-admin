# 텔레그램 알림 설정

크롤러가 신규 업체를 감지하면 `@bangjuclaw_bot` 채팅으로 자동 알림.

## 알림 타입

### 1. 신규 업체 감지
```
🆕 창원 새로오픈 맛집 3개 발견!

📍 가게A (한식) ★4.5
   리뷰 30건 | "음식 맛있어요" 25명
   창원 마산합포구 ...
   [네이버 지도 링크]
```

### 2. 크롤링 완료 요약
```
✅ 크롤링 완료 (2026-04-17 09:23)
총 50개 수집, 신규 0개
```

## 구성 요소

- **봇 토큰**: `~/.claude/channels/telegram/.env`에 저장
- **chat_id**: `914082906` (bangju)
- **구현**: `scripts/crawl-restaurants/src/notify.ts`

## 수동 테스트

```bash
curl -X POST "https://api.telegram.org/bot{TOKEN}/sendMessage" \
  -H "Content-Type: application/json" \
  -d '{"chat_id":"914082906","text":"테스트"}'
```

## 알림 조건

- 2회차 이상 크롤링에서만 "신규" 감지 (1회차는 모두 신규이므로 제외)
- DB에 이미 있는 `naver_place_id`와 비교하여 신규만 알림
