# 서버 및 프로그램 실행 순서

## 전체 실행 순서

### 1. OrbStack (Docker 엔진)
```bash
open -a OrbStack
```

### 2. Metabase + Directus 컨테이너
```bash
cd /Users/bangju/Documents/PROGRAM/unniepick-admin
docker compose -f docker-compose.bigdata.yml --env-file .env.bigdata up -d
```
- Metabase: http://localhost:3100
- Directus: http://localhost:8055

### 3. Next.js 개발 서버
```bash
cd /Users/bangju/Documents/PROGRAM/unniepick-admin
npm run dev
```
어드민: http://localhost:3001

### 4. (자동) launchd 크롤러
맥북 부팅/화면 열기 시 자동 실행.
```bash
# 상태 확인
launchctl list | grep unniepick

# 수동 실행
launchctl start com.unniepick.crawl-restaurants
```

## URL 맵

| 서비스 | URL | 용도 |
|--------|-----|------|
| 언니픽 어드민 | http://localhost:3001 | 메인 |
| 창원 맛집 목록 | /dashboard/restaurants | 카드 뷰 |
| 크롤링 키워드 | /dashboard/restaurants/keywords | 관리 |
| 맛집 분석 | /dashboard/restaurants/analytics | Metabase 임베드 |
| 문서 Wiki | /dashboard/docs | 기획/문서 |
| Metabase | http://localhost:3100 | BI |
| Directus | http://localhost:8055 | 데이터 관리 |

## 중지

```bash
# 컨테이너 중지
docker compose -f docker-compose.bigdata.yml --env-file .env.bigdata down

# 크롤러 자동실행 중지
launchctl unload ~/Library/LaunchAgents/com.unniepick.crawl-restaurants.plist

# Next.js: Ctrl+C
```
