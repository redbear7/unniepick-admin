# 일본 HotPepper Gourmet 성공 요인 분석

> 창원 AI 맛집 추천 시스템(V2)의 벤치마크로 선정된 일본 1위 맛집 예약 앱 HotPepper(ホットペッパーグルメ)의 핵심 성공 요인 분석.

## 시장 위상

- **리크루트(Recruit) 운영** — 2000년대부터 축적된 맛집 DB
- **2021년 기준 일본 최대 규모**:
  - 예약 가능 매장: **48,370개** (1위)
  - 사용자 만족도: **57.5%** (1위)
  - 사용 편의성: **62.5%** (1위)
- 주 사용자층: **20대 + 여성 비율 높음**

## 6대 차별화 요소

### 1. 🎯 シーン検索 (상황 검색) — 최대 강점
**「女子会、デート、歓送迎会、接待、合コン」** 등 *이용 목적/상황* 기반 검색.

예: "女子会"로 검색 → 분위기·메뉴·가격대가 여자 모임에 적합한 매장 자동 필터링.

**→ 언니픽 V2 적용**: "목적/상황" 태그를 12개 카테고리 중 최상위로 격상. 한국식 시나리오 추가 (상견례, 돌잔치, 회사MT, 프로포즈 등).

### 2. 🔥 こだわり条件 (구체 조건 필터)
- 음식 욕구 기반: "チーズが食べたい", "辛いものが食べたい"
- 업태 세분화: 和食 > 懐石·割烹 / 洋食 > イタリアン·フレンチ 등 3~4단 계층

**→ 언니픽 적용**: "음식 욕구" 카테고리 신설 ("매운거땡긴다", "치즈먹고싶다").

### 3. ⚡ 席押さえ (실시간 자리 선점)
- 현재 위치 주변 **지금 입장 가능한 매장** 실시간 표시
- 앱 내에서 즉시 자리 확보

**→ 언니픽 적용**: 네이버 테이블링 + 사장님 수동 토글 연동.

### 4. 💰 포인트·쿠폰 통합
- **Ponta / d포인트** 적립 및 사용
- 앱 내 쿠폰 즉시 표시 → 가게에서 바로 제시
- 재방문 유도 Lock-in 효과

**→ 언니픽 적용**: V1 매장 독점 쿠폰, 향후 포인트 시스템.

### 5. ✨ こだわり情報 (점주 자랑 섹션) — 감성 차별화
사장님이 직접 입력하는 감성 카테고리:
| 섹션 | 설명 |
|------|------|
| 店長のおすすめ料理 | 대표 메뉴 + 스토리 |
| こだわりの空間 | 공간 자랑 (인테리어/좌석 특징) |
| 心をこめたおもてなし | 서비스 자랑 (환영 프로토콜) |

**→ 언니픽 적용**: "점장 자랑" 카테고리 신설. V1 매장 사장님이 직접 입력 → AI 추천 시 "왜 추천" 근거로 인용.

### 6. 📊 점주 분석 도구 (B2B)
- 내 가게가 **어떤 키워드로 노출/유입됐는지** 분석
- 약점 파악 → 쿠폰/홍보 전략 제안

**→ 언니픽 적용**: 유료 B2B SaaS 모델로 확장 가능 (V1 구독료 외 추가 수익).

## HotPepper UX 패턴

### 홈 구조
```
- 현재 위치 기반 추천
- 오늘의 シーン 칩 (女子会/デート/歓送迎会/...)
- 새로 오픈한 매장
- 지금 바로 갈 수 있어요 (실시간)
- 이번 주 쿠폰
- 인기 검색어
```

### 검색 → 결과 플로우
1. 자연어 or シーン 칩 선택
2. 필터 (거리/가격/조건) 미세조정
3. 카드형 결과 (사진 + 점장 자랑 + 쿠폰 + 리뷰)
4. 원클릭 예약

## 창원 적용 시사점

1. **"シーン = 목적 태그"** 를 핵심 검색 축으로
2. **"こだわり = 구체 욕구"** 태그 추가로 즉흥 검색 유입 강화
3. **"점장 자랑"** 섹션은 V1 매장과 강력한 시너지
4. **실시간 대기 정보**는 작은 차별화로 큰 만족도
5. **B2B 분석 도구**는 V1 수익모델과 결합 가능
