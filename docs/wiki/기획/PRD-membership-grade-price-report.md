## 1. Summary

언니픽 앱에 현재 브론즈/실버/골드 3단계 레벨을 10단계 세분화 등급제로 개편하고, 사용자가 가게 메뉴 가격을 직접 제보하면 포인트·배지·혜택으로 보상하는 시스템을 도입한다. 이를 통해 사용자의 앱 재방문율과 콘텐츠 품질(가격 정확도)을 동시에 높인다.

## 3. Background

### 지금 무슨 상황인가?

언니픽은 창원 지역 뷰티·식음료 가맹점의 쿠폰과 혜택을 모아주는 앱이다. 지금은 브론즈 / 실버 / 골드, 3가지 레벨만 있다. 레벨이 3개뿐이라 사용자가 빨리 골드에 도달하고 더 이상 성장하는 느낌을 받지 못한다. 앱에 오래 머물 이유가 약하다.

최근 **가격 제보 기능(PriceReportScreen)** 이 개발 완료되었다. 사용자가 직접 가게 메뉴 가격을 입력할 수 있다. 그러나 현재 제보에 대한 **보상이 없다.** 보상 없이는 제보 참여율이 낮아 데이터 품질이 나빠진다.

### 왜 지금인가?

- 가격 제보 기능이 막 출시되어 사용자 보상 설계가 시급하다.
- Supabase `profiles` 테이블에 `level`, `level_progress`, `point_to_next` 컬럼이 이미 준비되어 있다.
- 초기 사용자를 락인(lock-in)할 최적의 시점이다.

## 6. Value Proposition

- 레벨이 오를수록 쿠폰 할인율·개수 혜택 증가
- 가격 제보 시 즉시 픽포인트 획득 (관리자 승인 후 확정)
- 투표(맞아요/틀려요)로도 소소한 포인트 적립
- 특별 배지(제보왕, 인증왕 등)로 자랑거리 생김

## 7. Solution

### 7.1 레벨 1~10 등급표

| 레벨 | 등급명 | 이모지 | XP 누적 | 주요 혜택 |
|------|--------|-------|---------|-----------|
| 1 | 새내기 언니 | 🌸 | 0 XP | 기본 쿠폰 접근 |
| 2 | 단골 언니 | ☕ | 100 XP | 단골 전용 쿠폰 +1장/월 |
| 3 | 핫플 언니 | 🔥 | 300 XP | 신규 가게 쿠폰 우선 알림 |
| 4 | 탐정 언니 | 🔍 | 600 XP | 가격 제보 보너스 XP +20% |
| 5 | 인싸 언니 | 💅 | 1,000 XP | 월 특별 쿠폰 팩 제공 |
| 6 | 리뷰 언니 | ✍️ | 1,500 XP | 리뷰 작성 시 추가 포인트 |
| 7 | 파워 언니 | ⚡ | 2,200 XP | 생일 쿠폰 2배 |
| 8 | 전문가 언니 | 🎓 | 3,000 XP | 가격 제보 즉시 반영 권한 |
| 9 | 레전드 언니 | 👑 | 4,000 XP | VIP 전용 한정 쿠폰 |
| 10 | 언니픽 마스터 | 💎 | 5,500 XP | 전 가게 10% 추가 할인 / 어드민 직접 연락 |

> **XP 감소 정책:** 6개월 비활동 시 레벨 1 강등 (유예 알림 1회 발송)

### 7.2 XP 획득 행동표

| 행동 | XP | 픽포인트 | 1일 최대 |
|------|-----|---------|---------|
| 쿠폰 다운로드 | 5 | 5P | 10회 |
| 쿠폰 사용 (인증) | 20 | 20P | 5회 |
| 가격 제보 (신규) | 30 | 30P | 5회 |
| 영수증 인증 | 40 | 40P | 3회 |
| 친구 초대 | 100 | 100P | 월 5회 |
| 출석 체크 (연속 7일) | 50 | 50P | 주 1회 |

### 7.3 가격 제보 보상 (2단계)

1. **제보 즉시** → 임시 포인트 +30P ('검토 중' 표시)
2. **관리자 승인 후** → 확정 포인트 +30P, XP +30 지급 / 거절 시 임시 포인트 취소

### 7.4 배지 시스템

퍼스트 제보 🥇 / 제보왕 📋 / 영수증 마스터 🧾 / 출석 여왕 📅 / 쿠폰 헌터 🎫 / 핫플 발굴자 📍 / 투표 달인 🗳️ / 친구 대장 👯 / VIP 💎

### 7.5 픽포인트 사용처

| 사용처 | 필요 포인트 |
|--------|-----------|
| 추가 쿠폰 슬롯 (+2장/당일) | 50P |
| 스탬프 카드 부스터 (2배) | 100P |
| 랜덤 쿠폰 뽑기 | 200P |
| 레벨업 부스터 (XP +50%, 24시간) | 300P |

### 7.6 DB 스키마 (신규)

