# V2 창원맛집 탭 UI 디자인 시안

> 언니픽 앱 V2에 추가될 창원맛집 탭의 화면 구성 시안. HotPepper + 언니픽 DNA 반영.

## 디자인 원칙

1. **대화 먼저, 필터 나중** — 자연어 검색이 주, 버튼 필터는 보조
2. **시나리오 중심** — "상견례", "데이트" 같은 상황 칩이 홈 상단
3. **로컬 깊이** — 창원/마산/진해 특화 콘텐츠 강조
4. **신뢰도 가시화** — AI가 왜 이걸 추천하는지 명확하게 노출
5. **언니픽 오렌지** (`#FF6F0F`) 액센트, 다크모드 기본

## 컬러 팔레트

| 용도 | 색상 | HEX |
|------|------|-----|
| 메인 액센트 | 언니픽 오렌지 | `#FF6F0F` |
| 배경 (다크) | 다크 그레이 | `#0F0F0F` |
| 카드 배경 | 차콜 | `#1A1A1A` |
| 텍스트 주 | 화이트 | `#FFFFFF` |
| 텍스트 부 | 라이트 그레이 | `#E8E8E8` |
| 텍스트 힌트 | 뮤티드 그레이 | `#888888` |
| 긍정 | 그린 | `#22c55e` |
| 경고 | 앰버 | `#f59e0b` |
| 오류 | 레드 | `#ef4444` |

## 타이포그래피

- 헤딩: Pretendard Bold 24/20/18/16px
- 본문: Pretendard Regular 14-15px, line-height 1.7
- 캡션: Pretendard 12px
- 숫자 강조: Pretendard SemiBold

## 컴포넌트 원칙

- **둥근 모서리**: 일관된 `rounded-xl` (16px) / 칩은 `rounded-full`
- **그림자 없음**: 다크모드는 border로 구분
- **애니메이션**: 150-250ms ease-in-out
- **터치 영역**: 최소 44×44pt
- **탭/입력 피드백**: ripple 또는 배경 변화
