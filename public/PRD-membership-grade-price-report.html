1Summary

언니픽 앱에 현재 브론즈/실버/골드 3단계 레벨을 10단계 세분화 등급제로 개편하고, 사용자가 가게 메뉴 가격을 직접 제보하면 포인트·배지·혜택으로 보상하는 시스템을 도입한다. 이를 통해 사용자의 앱 재방문율과 콘텐츠 품질(가격 정확도)을 동시에 높인다.

3Background

🏪 지금 무슨 상황인가?

언니픽은 창원 지역 뷰티·식음료 가맹점의 쿠폰과 혜택을 모아주는 앱이다. 지금은 브론즈 / 실버 / 골드, 3가지 레벨만 있다. 레벨이 3개뿐이라 사용자가 빨리 골드에 도달하고 더 이상 성장하는 느낌을 받지 못한다. 앱에 오래 머물 이유가 약하다.


최근 가격 제보 기능(PriceReportScreen)이 개발 완료되었다. 사용자가 직접 가게 메뉴 가격을 입력할 수 있다. 그러나 현재 제보에 대한 보상이 없다. 보상 없이는 제보 참여율이 낮아 데이터 품질이 나빠진다.

⏰ 왜 지금인가?
  • ✅ 가격 제보 기능이 막 출시되어 사용자 보상 설계가 시급하다.
  • ✅ Supabase profiles 테이블에 level, level_progress, point_to_next 컬럼이 이미 준비되어 있다. DB 인프라는 갖춰진 상태다.
  • ✅ 초기 사용자를 락인(lock-in)할 최적의 시점이다. 초기에 성장 루프를 만들어야 나중에 MAU를 높이기 쉽다.

7.3 기술 스펙 (DB)

profiles 테이블 — 신규 추가 컬럼
ALTER TABLE profiles ADD COLUMN total_xp integer DEFAULT 0, -- 전체 누적 XP point_balance integer DEFAULT 0, -- 현재 포인트 잔액 badge_ids text[] DEFAULT '{}' -- 획득한 배지 ID 목록
신규 테이블: xp_logs
CREATE TABLE xp_logs ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), user_id uuid REFERENCES profiles(id), action text, -- 'price_report' | 'coupon_use' | 'vote' … xp_amount integer, point_amount integer, created_at timestamptz DEFAULT now() );
신규 테이블: price_reports
CREATE TABLE price_reports ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), user_id uuid REFERENCES profiles(id), store_id uuid REFERENCES stores(id), menu_name text, price integer, status text DEFAULT 'pending', -- 'pending' | 'approved' | 'rejected' reject_reason text, created_at timestamptz DEFAULT now() );

8Release

PHASE 1 · 약 3~4주
핵심 MVP — 레벨 시스템 기반 + 가격 제보 보상 연동
  • DB 스키마 추가 (xp_logs, price_reports, profiles 컬럼)
  • XP 적립 로직 (Supabase Edge Function 또는 앱 내 처리)
  • 레벨 계산 함수 (total_xp → level, level_progress, point_to_next)
  • 마이페이지: 레벨 바 + 포인트 잔액 표시
  • 가격 제보 승인 시 XP + 포인트 지급 (어드민 승인 액션)
  • 레벨업 푸시 알림
PHASE 2 · 약 2~3주
배지 + 투표 보상
  • 배지 지급 로직 (조건 달성 시 자동 지급)
  • 마이페이지 배지 그리드 UI
  • 투표 참여 XP 반영
  • 제보왕 배지 자동 판정 (월말 집계)
PHASE 3 · 약 2~3주
포인트 사용처 + 레벨업 애니메이션
  • 포인트 리딤 기능 (쿠폰 슬롯, 랜덤 뽑기 등)
  • 레벨업 축하 애니메이션 (Lottie / Reanimated)
  • XP 로그 히스토리 화면
  • 6개월 비활동 강등 자동화 (Cron)
PHASE 4 · 출시 후 피드백 반영
고도화
  • 레벨 난이도 조정 (XP 기준값 튜닝)
  • 시즌 이벤트 (기간 한정 배지, XP 2배 이벤트)
  • 친구 초대 보상 강화
  • 등급별 전용 쿠폰 큐레이션 자동화
