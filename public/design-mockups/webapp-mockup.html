🔐 인증 방식
SMS OTP
→ Supabase signInWithOtp({ phone })
→ 솔라피 SMS 발송

→ Supabase OAuth
signInWithOAuth({ provider: 'kakao' })
→ /auth/callback 콜백 처리
→ 기존 콜백 라우트 재사용 가능
📂 Next.js 라우트 구조
app/
├── page.tsx        ← 홈
├── login/          ← 로그인
├── stores/

├── wallet/         ← 쿠폰함
├── stamps/         ← 스탬프
└── my/            ← 마이페이지
